# 📊 Order Database Practice 2025 – Excel Data Cleaning & Formatting Project

## 📖 Project Overview

This project demonstrates practical Excel data cleaning, transformation, and formatting techniques performed on the **Order Database Practice 2025** dataset. The objective was to improve data quality, ensure consistency, and prepare the dataset for further analysis and reporting.

## 🎯 Project Goals

- Identify and remove duplicate records.
- Handle missing values effectively.
- Standardize text formats and correct data inconsistencies.
- Transform and restructure data for better usability.
- Apply formatting techniques to improve readability and visualization.

---

## 🛠 Tasks Performed

### 1. Duplicate Data Removal
- Identified duplicate rows based on the entire row content.
- Removed duplicate records to maintain data accuracy and integrity.

### 2. Missing Value Handling
- Checked for missing values in the **Price** column.
- Calculated the average price and used it as a reference for handling missing data.
- Identified missing values in the **Product Category** column.
- Filled missing category values based on the corresponding **Product Name**.

### 3. Data Standardization
- Detected inconsistent text formats and typographical errors in:
  - Product Name
  - Category
- Corrected inconsistencies using Excel's **Find and Replace** functionality.

### 4. Data Transformation
- Split the **Product ID** column into two separate columns:
  - Manufacturing Date
  - Country Code
- Removed unnecessary characters during the transformation process.

### 5. Column Merging
- Combined the **Brand Name** and **Product Name** columns.
- Created a new column named **Product Brand**.

### 6. Data Formatting
- Converted the **Price** column to **Currency Format**.
- Formatted the **Manufacturing Date** column to display dates in:

```text
DD-MM-YYYY
```

### 7. Conditional Formatting
- Applied **Data Bar Conditional Formatting** to the **Price** column for visual comparison of values.
- Created a custom conditional formatting rule in the **Category** column to highlight cells where:

```text
Category = Electronics
```
